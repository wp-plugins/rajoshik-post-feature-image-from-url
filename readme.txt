This Plugin will allow user to upload post feature image from image url. Image will be added as post feature image..

== Description ==
Rajoshik Post Feature Image From URL is a Plugin for Wordpress. This Plugin will be give option to user set post feature image from any image URL. No Need to upload image just put the image url and publish the post, image will be automatically upload with post from the url as post feature image. 



== Installation ==

- Install from WordPress admin panel >> Plugins >> Add new plugin >> Search for `Rajoshik Post Feature Image From URL' plugin. After you install, activate the plug-in. 

- After Successfully Install & Active.
